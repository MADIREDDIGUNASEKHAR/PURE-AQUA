# 💧 RK Pure Aqua — Landing Page

A fully interactive MERN Stack + Tailwind CSS landing page for **RK Pure Aqua**, featuring stunning animations, smooth transitions, and a complete UI.

## 🚀 Quick Start

### Prerequisites
- Node.js v16+ installed
- npm v8+

### Install & Run

```bash
# Step 1 — Install all dependencies
npm run install-all

# Step 2 — Start both server and client
npm run dev
```

The app will open at:
- **Frontend**: http://localhost:3000
- **Backend API**: http://localhost:5000

## ✨ Features

- **MERN Stack** — MongoDB-ready Express backend + React frontend
- **Tailwind CSS** — Fully responsive utility-first styling
- **Animated SVG Bottles** — Pure custom SVG water bottle illustrations
- **Floating Bubble Particles** — Dynamic JS particle system
- **Smooth Scroll** — Section-by-section navigation
- **Intersection Observer** — Reveal animations on scroll
- **API Integration** — Products and enquiry form via Express
- **Glass Morphism UI** — Backdrop blur card design
- **Shimmer Text Effects** — CSS animated gradient text
- **Wave Animations** — Layered SVG wave effects
- **Mobile Responsive** — Hamburger menu + responsive grid
